1 Bayes theorem

We can use the conditional prob. which computes probabilities given that a certain event happened. This rules can be applied with the addition or multiplication rules, for instance. Consider the example on the slides, we can derive the Law of totla probability. 

P(A|B)= P(A \cub B) / P(B) = (P(B|A)*P(A)) / (P(B|A)*P(A) + P(B|~A)*P(~A))

The last rule is what Bayes theorem is about.

Example: we have .1 of the population has a disease. We have a medical test: has a disease => positive test (result with prob. .98) And no disease => negative test (0.99).

Now suppose that X takes the test, the result is positive. What is the prob. that X has the disease given the positive test outcome?

Let B = {positive} and A ={disease}. We want to compute P(A|B).

Although we know that P(B|A) = .98, but it's not the same as P(A|B). Thus we need to use Bayes theorem.

P(A|B) = (.98 * .1) / (.98 * .001 + .01 * .999) is approx. .089 So X has a prob. of 8.9% of having the disease. (Assign numbers for the population to make the example clearer.)

1.1 Note

The probability P(A|B) is not the same as P(B|A). This confusion might lead to unforeseen, or even horrible, interpretations. For example, Sally Clark was convicted for killing her two babies that were born two years after each other and both died at age of almost 3 months. What a professor calculated at the time is P(Babies died | Clark's innocent) which was estimated to be 1 over 70 million!! However, it should have been P(Clark's innocent | Babies Died) which would have resulted in much smaller prob.

1.2 Partition

Events A1,…,Am are called a partition if

  • pairwise disjoint: Ai ∩ A+j = ∅, if i != j.
  • union is entire sample space: A1 ∪ A2,,…, ∪ Am = Ω.

Moreover, we can then calculate Bayes theorem for an event with the whole partition: P(B) = P(B ∩ Ai) which is the same as P(B|Ai)*P(Ai)

Similarly, we can calculate some element r in Ω: P(Ar|B) = (P(B|Ar)*P(Ar)) / (P(B|Ai)*P(Ai))

Note the last term is the law of total probability

2 Probability Distributions

A random variable represent the numerical value to each outcome of a prob. experiment. Ex Ω = { 1,3,5 } when throwing a die and getting an old number. You can say then X(1)=1,… And now we can ask P(X=1)=?

Definition: probability distribution denotes all possible values of a random variable.

Moreover, there are two random variables (in this course at least):

  1. Discrete Random variable: has finite values. Its distribution is the collection of all their individual probabilities.

    1. Consider toss two fair coins, And X is the number of heads; ie X(HH) = 2, X(HT)=1, X(TH)=1, X(TT)=0, what is the probability distribution of X?

      I.e. P(X=0), P(X=1), P(X=2). Which are the values: 1/4, 1/2, and 1/4 respectively.

  2. Continuous random variable: has uncountably different values. Its distribution is calculated by the density function

2.1 Expected values (mean)

The expected value of a discrete random variable is denoted by E and is the mean of the mean value of the outcomes of that variable.

The mean is denoted by μ and is calculated as: ∑(X*P(X)).

2.2 Parameters of probability distribution

When considering parameters of probability distribution we talk about a population and not a sample. And of these population we can derive: mean, variance and *standard deviation.

  1. mean (denoted by μ): is the central or average value (of the random variable)
    1. ∑[X*P(X))
  2. variance and standard deviation measure the variation in the random variable
    1. variance:u∑[(X-μ)2 * P(X)]
    2. SD: \sqrt{variance,}.

Moreover, these parameters are useful to determine whether the value of the random variable is unusually high or low. There are two ways to find the maximum and minimum values:

  1. range rule of thumb: if the value is 2 standard deviation away from the mean, if so we say the value of the random variable X lies outside of the limits.
    1. min= μ - 2*SD and max = μ + 2*SD
    2. E.g. is 2 girls an unusually high number of two births? (where x was 0, 1, 2 girls each with 0.25, 0.50, and 0.25 prob. respectively). After finding the min and max (-0.4 and 2.4) we can say that 2 girls is not an unusually high value of X (i.e. # of girls).
    3. This rule however does not provide a rigid value, thus 3 could also be used.
  2. with probability: we can find for a random variable an unusually high or low by calculating for P(X or more) or P(X or fewer) respectively. If the result is <= 0.05, we could conclude that the probability of X to happen (in comparison to some usual results) is high or low (based on the question). Thus if it is unusually high, we could say that the experiment that produced that unusual value is very likely to be sound, and the opposite (ie for unusually low) holds too.

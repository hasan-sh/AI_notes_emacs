1 Correlation

With statistical methods we cannot investigate the causality between events, but rather the relationship between them. E.g. if obtaining higher assignment grades has a relation with getting a higher exam grade, we cannot say that the one causes the other. 

The test statistic of a correlation test is always *two-tailed*. (unless we want to check for positive/negative correlation)
Moreover, the null hypothesis is always: \rho = 0 versus \rho != 0.

1.1 setup and scatter plot

  • question whether there is a relationship between x and y.
  • experiment: measure the variables of n subjects.
  • sample n pairs: (x1,y1)….(xn,yn).
    • this is not the same as matched pairs testing.
  • further, the first step in identifying a relationship between variables is through a scatterplot.
  • linear relationship; positive or negative.
  • non-linear relationship; positive or negative.

  • no relationship; random data. 

    we study linear relationship in this course.

1.2 definitions

Correlation means having an association between variables. When measuring the values of V1 and V2:

  1. higher V1 values associated with higher V2 -> positive correlation

  2. higher V1 values associated with lower V2 -> negative correlation

    Moreover, linear correlation simply indicates that the plotted points follow some straight line.

1.3 How?

The formula that calculates the sample variance of some population is used to calculate correlation. However, here we call it linear correlation coefficient, denoted by ρ (Greek letter Rho). Moreover, the simple linear correlation coefficient is denoted by the letter r. and it is calculated similar to the variance, but taking the two dataset into account. 

r = 1/n−1 * ∑^n_(i=1)(x_i−X)(y_i−Y)/s_xs_y.  X and Y means of the samples. S_x and S_y are the SD of the samples.
This formula does not have to be memorized.

We can calculate the r using R: r: cor(x, y).

IMPORTANT NOTES

  • if r is close to -/+1, then there is a linear correlation, if not then there is not (i.e. if close to 0).
  • Furthermore, the correlation is sensitive to outliers
    • i.e. one outlier may make the r value very close or far from 0 (or 1).
    • we can exclude outliers, BUT always explain why!

The value of r is always between -1 and 1:

  • r= 1: perfect positive linear relationship;
  • r>0: positive linear relationship
  • r≈0: no linear relationship;
  • r<0: negative linear relationship
  • r=−1: perfect negative linear relationship.

IMPORTANT Correlation does not imply causality. Because of:

  • Third-variable problem: unmeasured variable that effects both variables.E.g. ice cream sales and murder rate are correlated. Third variable: sunshine.
  • Direction of causality: correlation coefficient does not tell which variable could cause the other to change. E.g. wind mill rotation and wind speed are correlated, but wind mills do not cause the wind.
correlation tests are always two-tailed

2 Regression

The goal of linear regression is to find the (approximate) straight line of the data points. The line is denoted by: ^y = b0 + b1x. This is called the fitted regression equation. Where b0 is the intercept and b1 is the slope. 

x is called the explaintory variable and y is the output/response variable

Moreover, we calculate the slope (ie b1) by: b1 = r * sy / sx (s is the standard deviation of each. And the intercept is calculated by: b0 = Y - b1X (where Y and X are the sample means.)

In R we use lm (i.e. linear model), e.g. lm(y~x, data=.._), to estimate the regression line.

2.1 Appropriate linear model?

  • regression equation should be a reasonable fit (qualitative check using plot)
  • Coefficient of determination r2 is large!

  • Linear relationship is significant (test the claim B1 = 0 against B0 != 0).

    The slope (i.e. b1) represents the marginal change of y. That is if x increases by 1, y will be increased by b1.

The r2 is calculated by: Sx2 / (Sx2 + Sy2), where S() is the SD of each. It is between 0 and 1, if r2 is large (close to 1), then the linear model is appropriate.

The test statistic would then be: Tβ = b1 - β1 / Sb1. Calculate the P-value and check if it is less than the significant level (or with the critical region; whether it is within it)

Further, for testing linearity we have to check if the errors are:

  1. independent; assumed true (hard to check)
  2. normal dist: create normal QQ plot of the residuals
  3. Fixed SD; create residual plot!

The residual is the difference between the observed/true value yi and predicted value (i.e. estimated on the line): residuali = yi - ~yi = yi - (b0 + b1xi). We get the residual value from R by: lm(…)$res. The residual plot should be random and follows no obvious pattern. 

the residuals are based on the vertical distances between the actual data point and the regression line.

If the previous requirements are met, then the test is valid.

Furthermore, we can use prediction intervals to get an interval estimate of the value of y. We use prediction intervals; to get an interval estimate of the true value of y (for some given value x substituted in the regression equation)
